Создай менеджер задач со следующими возможностями:

1. Добавление задач через форму ввода
2. Удаление задач с анимацией исчезновения
3. Редактирование по двойному клику с всплывающей формой
4. Автоматическая нумерация позиций
5. Синхронизация с базой данных MySQL

Технические требования:
- Бекенд на Node.js с чистыми модулями http
- Frontend на чистом HTML/CSS/JavaScript
- Стили в современном минималистичном дизайне
- Адаптивная верстка
- Анимации для всех действий пользователя
- Обработка ошибок с уведомлениями

Особые пожелания:
- Плавные переходы при взаимодействии
- Подсветка активных элементов
- Валидация ввода
- Поддержка горячих клавиш (Enter/Escape при редактировании)
- Кросс-браузерная совместимость

Предоставь полный код сервера и клиента в двух файлах (index.js и index.html), готовый к запуску после установки зависимостей.
